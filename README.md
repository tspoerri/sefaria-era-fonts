# sefaria-era-fonts

Prototype: a fork/clone of Sefaria's source sheet builder where each source
renders in a Hebrew font matching the era in which it was written.

Font-to-era mapping is based on this research chart:
https://claude.ai/code/artifact/445bc177-627c-4dd7-b7e7-12163a84dfb7

Status: scaffolding in progress via an overnight cloud routine. See HANDOFF.md
for current state and next action.
